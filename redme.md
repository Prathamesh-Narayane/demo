Basic Git Commands in Terminal (Windows)
Since you're working in VS Code on Windows with the folder demogithub open, here are some common Git commands to run in the integrated terminal. Open the terminal in VS Code (Ctrl + `) and navigate to your folder if needed.

Initialize a new Git repository (if not already done):

Check repository status:
git status

Add files to staging:
git add.

Commit changes:
git commit -m ""

View commit history:
git log --oneline
